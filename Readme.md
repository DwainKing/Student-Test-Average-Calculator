# Student Test Score Average Calculator

A simple project that calculates the average of student test scores using both:
- C++ 
- Web version (HTML, CSS, JavaScript)

This project demonstrates the same programming logic implemented in two different environments.
---
## How to Use

### Web Version

1. Open the live demo link:  
   https://github.com/DwainKing/Student-Test-Average-Calculator.git

2. Enter student scores in the input box  
   Example: `90, 85, 78`

3. Click the **Calculate** button

4. View the average score displayed on screen

---

### C++ Version

1. Open `cpp-version/main.cpp`
2. Compile using any C++ compiler
3. Run the program
4. Enter number of students
5. Enter each test score
6. View the calculated average

---

## What This Project Demonstrates

- Loops (for iteration)
- Accumulator variables
- Input/output handling
- Basic algorithm design
- Same logic implemented in multiple languages
